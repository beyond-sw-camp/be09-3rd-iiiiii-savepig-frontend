## D. 실행(서버 구동)
- Vue 프로젝트 실행: `npm run dev`
- json server 실행: `json-server --watch (프로젝트 root)/src/data/mockup.json --port 8080`
- 절약지도 화면은 카카오 지도 API를 사용하고 있습니다. 지도 화면을 확인하기 위해서는 `.env.local` 파일에 API key값을 추가해야 합니다.
